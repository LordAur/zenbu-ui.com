## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/icon" />

## Usage
Our icon component using [Heroicons](https://heroicons.com/) from Tailwind. You can search icon name from heroicons.

```jsx
import { Icon } from '@zenbu-ui/icon'

const App = () => {
  return(
    <>
      <Icon icon="academic-cap" color="blue" colorContrast="600" />
      <Icon icon="academic-cap-solid" color="blue" colorContrast="600" />
    </>
  )
}

export default App
```
