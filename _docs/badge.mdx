## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/badge" />

## Usage
```jsx
import { Badge } from '@zenbu-ui/badge'

const App = () => {
  return(
    <Badge count={5}>
      <YourComponent />
    </Badge>
  )
}

export default App
```

### Overflow Count
<PreviewBadgeOverflow />

```jsx
<Badge count={100} overflowCount={99}>
  <YourComponent />
</Badge>
```

### Preffix and Suffix
<PreviewBadgePreSuff />

```jsx
<>
  <Badge count={100} preffix=">">
    <YourComponent />
  </Badge>
  <Badge count={100} suffix="K">
    <YourComponent />
  </Badge>
</>
```
