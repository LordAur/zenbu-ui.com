## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/input" />

## Usage

```jsx
import { Input } from '@zenbu-ui/input'

const App = () => {
  return(
    <Input.Phone name="tester" placeholder="Phone number" />
  )
}

export default App
```

### Change Country Code
By default, the country code is ID (Indonesia), but you can change the default country code with use defaultCountry property.

<PreviewPhoneCountry />

```jsx
<Input.Phone defaultCountry="US" name="tester" placeholder="Phone number" />
```
