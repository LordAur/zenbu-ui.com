## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/avatar" />

## Usage
```jsx
import { Avatar } from '@zenbu-ui/avatar'

const App = () => {
  return(
    <Avatar alt="Avatar" src="https://" />
  )
}

export default App
```

### Text
You can change the avatar background color and the text color using property **bgColor**, **bgColorContrast**, **darkBgColor**, and **darkBgColorContrast** to change the background color.

Use property **textColor**, **textColorContrast**, **darkTextColor**, and **darkTextColorContrast** to change the text color.

<PreviewAvatarText />

```jsx
<Avatar
alt=""
bgColor="yellow" bgColorContrast="200"
textColor="yellow" textColorContrast="500"
text="YP" />
```
