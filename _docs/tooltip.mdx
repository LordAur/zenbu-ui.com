## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/tooltip" />

## Usage

```jsx
import { Tooltip } from '@zenbu-ui/tooltip'

const App = () => {
  return(
    <Tooltip
    trigger={(
      <Button>Tooltip Button</Button>
    )}
    content={(
      <p>Bottom tooltip</p>
    )} />
  )
}

export default App
```

### Popup
If you want to make your tooltip dialog not immediately closed, used the **popup** property.

<PreviewTooltipPopup />

```jsx
<Tooltip
popup
trigger={(
  ...
)}
content={(
  ...
)} />
```
