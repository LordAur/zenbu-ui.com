## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/statistic" />

## Usage

```jsx
import { Statistic } from '@zenbu-ui/statistic'

const App = () => {
  return(
    <Statistic title="Active Users" value={1000} />
  )
}

export default App
```
