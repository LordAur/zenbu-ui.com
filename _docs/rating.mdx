## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/rating" />

## Usage
```jsx
import { Rating } from '@zenbu-ui/rating'

const App = () => {
  return(
    <Rating defaultValue={3} color="blue" colorContrast="600" />
  )
}

export default App
```

### Count
You can set the max rating with count property:

<PreviewRatingCount />

```jsx
<Rating defaultValue={3} count={6} color="blue" colorContrast="600" />
```

### Heart
<PreviewRatingHeart />

```jsx
<Rating defaultValue={3} heart color="pink" colorContrast="500" />
```
