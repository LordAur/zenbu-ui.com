## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/message" />

## Usage

```jsx
import { Message } from '@zenbu-ui/message'

const App = () => {
  return(
    <Message>
      This is a message
    </Message>
  )
}
```
