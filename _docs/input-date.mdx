## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/input" />

## Usage

```jsx
import { Input } from '@zenbu-ui/input'

const App = () => {
  return(
    <Input.Date name="tester" onChange={(unix, date) => {...}} />
  )
}

export default App
```

### Format
<PreviewInputDateFormat />

```jsx
<Input.Date
format="DD MMM YYYY"
name="tester" onChange={(unix, date) => {...}} />
```

### Time
<PreviewInputDateTime />

```jsx
<Input.Date
time format="DD/MM/YYYY HH:mm"
name="tester" onChange={(unix, date) => {...}} />
```
