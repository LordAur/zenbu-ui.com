## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/progress" />

## Usage

```jsx
import { Progress } from '@zenbu-ui/progress'

const App = () => {
  return(
    <Progress percentage={50} />
  )
}

export default App
```

### Coloring
<PreviewProgressColoring />

```jsx

color="red" colorContrast="400"
bgColor="slate" bgColorContrast="500"
```
