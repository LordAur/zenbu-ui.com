## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/popover" />

## Usage

```jsx
import { Button } from '@zenbu-ui/button'
import { Popover } from '@zenbu-ui/popover'

const App = () => {
  return(
    <Popover
    trigger={(
      <Button>Popover Button</Button>
    )}
    content={(
      <p>Bottom popover</p>
    )} />
  )
}

export default App
```

### Position
You can set the position of popover on **left**, **right**, **top** or **bottom**.

<PreviewPopoverPosition />

```jsx
<Popover
position="top"
trigger={( ... )}
content={( ... )} />
```
