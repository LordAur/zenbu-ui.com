## Font size
Control the font size of a component using the **fontSize** properties.

<FontSizeExample />

```jsx
<Text fontSize="sm"> ... </Text>
<Text fontSize="base"> ... </Text>
<Text fontSize="lg"> ... </Text>
<Text fontSize="xl"> ... </Text>
<Text fontSize="2xl"> ... </Text>
```

## Font weight
Control the font weight of a component using the **fontWeight** properties.

<FontWeight />

<FontWeightExample />
