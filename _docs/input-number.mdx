## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/input" />

## Usage

```jsx
import { Input } from '@zenbu-ui/input'

const App = () => {
  return(
    <Input.Number name="tester" placeholder="Place a number here" />
  )
}

export default App
```

### Separator
<PreviewInputNumberSeparator />

```jsx
<Input.Number
separator="dot"
name="tester" placeholder="Place a number here" />
```

### Currency
<PreviewInputNumberCurrency />

```jsx
<Input.Number
separator="dot"
currency currencyText={(
  <Text span fontSize="sm">
    Rp
  </Text>
)}
name="tester" placeholder="Place a number here" />
```

### Keyboard
If you used **keyboard** property, so users can increase  or decrease the value with arrow keys.

<PreviewInputNumberKeyboard />

```jsx
<Input.Number keyboard min={0}
separator="dot"
name="tester" placeholder="Place a number here" />
```
