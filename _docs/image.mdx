## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/image" />

## Usage
```jsx
import { Image } from '@zenbu-ui/image'

const App = () => {
  return(
    <Image alt="..." src="https://..." />
  )
}

export default App
```
