## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/grid" />

## Usage
```jsx
import { Grid } from '@zenbu-ui/grid'

const App = () => {
  return(
    <Grid>
      <Grid.Column width="1/2">...</Grid.Column>
      <Grid.Column width="1/2">...</Grid.Column>
    </Grid>
  )
}

export default App
```

### Width
You can set the width of components with percentage or with a fixed number. Read the [sizing](/docs/sizing) documentation, to know the available width percentage and fixed numbers.

<PreviewGridWidth />

### Nested
You can nest columns to have more flexibility in your design. You only need to follow this structure:

<PreviewGridNested />

```jsx
<Grid>
  <Grid.Column width="full">
    <YourComponent />
    <Grid.Column nested py="5">
      <Grid.Column width="1/2">
        ...
      </Grid.Column>
      <Grid.Column width="1/2">
        ...
      </Grid.Column>
    </Grid.Column>
  </Grid.Column>
</Grid>
```
