## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/card" />

## Usage
```jsx
import { Card } from '@zenbu-ui/card'

const App = () => {
  return(
    <Card
    cover="https://..." coverAlt="..."
    title={( ...  )}
    description={( ... )} />
  )
}

export default App
```

### Aspect Ratio
You can customize the aspect ratio of the cover image. By default, card components use a **3/1.464** aspect ratio, but if you want to use a **1/1** aspect ratio, use **square** or **video** to use a **16/9** aspect ratio.

<PreviewCardAspectRatio />

```jsx
<Card
cover="https://..." coverAlt="..."
coverAspectRatio="square"
title={( ...  )}
description={( ... )} />
```
