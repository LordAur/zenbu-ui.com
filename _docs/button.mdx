## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/button" />

## Usage
```jsx
import { Button } from '@zenbu-ui/button'

const App = () => {
  return(
    <Button color="blue" colorContrast="600">Button</Button>
  )
}

export default App
```
