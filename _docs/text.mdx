## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/typography" />

## Usage
To styling text we have a lot of properties, you must read the [font](/docs/font) documentation to understand it better.

```jsx
import { Text } from '@zenbu-ui/typography'

const App = () => {
  return(
    <Text
    color="slate" colorContrast="700"
    darkColor="slate" darkColorContrast="200"
    fontWeight="bold" textTransform="capitalize">
      The quick brown fox jumps over the lazy dog.
    </Text>
  )
}

export default App
```

### Indent
<PreviewTextIndent />

### First Letter
<PreviewTextFirstLetter />

```jsx
<Text
color="slate" colorContrast="700"
darkColor="slate" darkColorContrast="200"
firstLetterFontSize="7xl" firstLetterFontWeight="bold" firstLetterTextTransform="uppercase"
firstLetterTextColor="black" firstLetterSpacing={{
  mr: "2"
}}>
  ...
</Text>
```

### First Letter Float
<PreviewTextFirstLetterFloat />

```jsx
<Text
color="slate" colorContrast="700"
darkColor="slate" darkColorContrast="200"
firstLetterFontSize="7xl" firstLetterFontWeight="bold" firstLetterTextTransform="uppercase"
firstLetterFloat="left" firstLetterTextColor="black"
firstLetterSpacing={{
  mr: "2"
}}>
  ...
</Text>
```

### Decoration
<PreviewTextDecoration />

```jsx
<Text underline
textDecorationColor="pink" textDecorationColorContrast="500"
textDecorationStyle="wavy">
  The quick brown fox jumps over the lazy dog.
</Text>
```

