## Installation
Install the component from your command line.

<NpmInstall packageName="@zenbu-ui/separator" />

## Usage

```jsx
import { Separator } from '@zenbu-ui/separator'

const App = () => {
  return(
    <>
      <h2>Title was Here</h2>
      <Separator />
    </>
  )
}

export default App
```

### Coloring
You can coloring the border separator with property **borderColor** and **borderColorContrast**, read the [color palette](/docs/color-palette) documentation for more information.

<PreviewSeparatorColoring />

```jsx
<>
  <h2>Title was Here</h2>
  <Separator borderColor="blue" borderColorContrast="700" />
</>
```

### Style

<PreviewSeparatorStyle />

```jsx
<>

  <Separator borderWidth="2" borderStyle="dashed" />
</>
```

### Text
